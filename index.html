Introduction to CSS Styling

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and positioning of HTML elements.

Key Properties

  • Color and background-color
  • Font-size and font-family
  • Margin and padding
  • Border and border-radius

The CSS Box Model

Every HTML element is a box with four areas:

Content

The actual content (text, images, etc.)

Padding

Space between content and border

Border

The edge of the box

Margin

Space outside the border
